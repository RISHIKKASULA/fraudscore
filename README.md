# fraudscore

A calibrated card-fraud scoring service that makes expected-cost decisions — review a
transaction when p̂ × amount exceeds the cost of reviewing it — served over FastAPI,
evaluated in dollars with confidence intervals, not in accuracy.

> **Status: under construction.** Measured results, quickstart, and limitations land with
> the v1.0.0 release. Design: [docs/architecture.md](docs/architecture.md).
